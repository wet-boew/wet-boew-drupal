WetKit WYSIWYG
================
Integrates a WYSIWYG into Drupal

Key Features
* The best TinyMCE WYSIWYG in Drupal
* Addition of Kitchen Sink and Spellcheck Plugins
* A good MarkitUp Editor for Drupal
* Switching WYSIWYG + HTML text formats
